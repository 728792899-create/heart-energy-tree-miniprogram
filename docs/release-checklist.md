# Release Checklist

## 代码与公开基线

- [ ] 全新克隆、无 private config、无 `motion-studio/out` 时 `npm test` 全绿且测试数不少于 203。
- [ ] `npm run check:shared`、`check:syntax`、`check:cloud-deps`、`check:budgets` 通过。
- [ ] Remotion compositions 和独立 still smoke 通过；`node_modules/`、`.cache/`、`out/` 未进入 Git。
- [ ] GitHub Actions 三个 job 在无微信账号/无私有配置环境通过。
- [ ] buildTag 在客户端、云函数、README 一致。

## 安全与数据

- [ ] 可信 OPENID、角色鉴权、事务、幂等、内容安全与共享同步测试通过。
- [ ] 客户端对所有业务集合无写权限；实时消息只读规则限定 `auth.openid`。
- [ ] 集合、索引、规则、备份、恢复和版本迁移按文档核对。
- [ ] 发布前备份可解析、计数与校验值已记录；回滚版本兼容 state version 4。
- [ ] 仓库与发布证据不含私人数据或秘密。

## 体验与内容

- [ ] 小字号/浅色文字、88rpx 触控、窄屏、键盘、弱网、加载、重试和空状态通过。
- [ ] 远程 MP4→poster→原生三级降级和每级失败路径通过。
- [ ] 简化动效与声音关闭都持久化；系统静音被遵守。
- [ ] 合法文字/图片不误拦，风险回调闭环和幂等证据齐全。
- [ ] ImageGen/设计源/Remotion 素材都在预算内，无水印、无私人照片、无版权不明新增物。

## 产品边界

- [ ] 仍为固定两人私人版，没有公共注册、租户选择或陌生人匹配。
- [ ] 心愿金/奖励只做内部记账和线下手动兑现；不接真实支付，没有微信支付、充值、提现 API 或金融收益文案。
- [ ] 解绑、导出、删除、照片生命周期、授权撤回和异常账号恢复说明可访问，未把人工流程伪装成自动功能。

## 外部验收

- [ ] 微信开发者工具编译记录完成。
- [ ] 获授权后云函数部署、真实 buildTag 核对完成。
- [ ] 获授权后双账号真机表完成。
- [ ] 未完成项明确标注“外部权限阻塞”，附责任人和下一步；没有用 web-demo/本地测试冒充。
